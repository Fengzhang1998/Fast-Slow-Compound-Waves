# README

## MATLAB Code for Epileptiform Fast–Slow Compound Wave Model

This repository contains the MATLAB code used to reproduce the main simulation results presented in the manuscript:

**Title:**
**Epileptiform Fast-Slow Compound Waves in Neural Networks: Mechanisms of Dendritic Oscillations and Non-Synaptic Recruitment**

**Author:**
Feng Zhang

**Contact Email:**
[fengzhang1998@tju.edu.cn]


---

## Requirements

* MATLAB (R2022a or later recommended)

---

## How to Run

### 1. Main Simulations

To reproduce the main simulation results in the paper, run:

```
simulation_main.m
```

This script generates the primary simulation data for the epileptiform fast–slow compound waves under ephaptic coupling.

---

### 2. Data Visualization and Figure Reproduction

To reproduce specific figures from the manuscript, run:

```
data_plot.m
```

This script generates the following figures:

#### Macroscopic Local Field Potential (LFP) Simulation Results

* Figure 2a
* Figure 2b
* Figure 2d

These figures show the simulated macroscopic LFP dynamics.

#### Microscopic Membrane Potential and Endogenous Electric Field

* Figure 3a
* Figure 3c

These figures present:

* Membrane potential dynamics
* Endogenous electric field dynamics
